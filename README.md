TechGadget Comparison

📌 Project Overview

TechGadget Comparison is a simple web page that compares two gadget models: Standard Model and Pro Model. The page displays the features, prices, and shipping information of each model to help users easily see the differences.

🎯 Project Purpose

The goal of this project is to practice creating a clean and structured web layout using HTML and CSS. It demonstrates how product comparison sections are designed on modern websites.

🛠️ Technologies Used

- HTML5 – Used to structure the webpage content.
- CSS3 – Used to style the page, including layout, borders, spacing, and design.

📱 Features

- Two product comparison cards
- Clear pricing display
- Feature lists for each model
- "Buy Now" buttons
- Responsive and simple design

💻 Product Details

Standard Model

- Price: $99
- OLED Display
- 6-hour Battery
- Water Resistant
- 4G Enabled
- Shipping fee required

Pro Model

- Price: $199
- OLED Display
- 12-hour Battery
- Water Resistant
- 5G Enabled
- Free Shipping
